# AlexMusic 🎵

AlexMusic é um player de mídia moderno e minimalista para Android, desenvolvido com as tecnologias mais recentes do ecossistema Google. Focado em performance e estética, o app oferece uma experiência de áudio e vídeo fluida com um visual futurista.

---

## ✨ Funcionalidades

- **🎧 Reprodução em Segundo Plano**: Graças à integração com `MediaSessionService` e `Media3`, sua música nunca para, mesmo com o app minimizado.
- **📱 Controles no Sistema**: Gerencie sua playlist diretamente pela barra de notificações ou tela de bloqueio com suporte total a controles de mídia do Android.
- **📊 Equalizador Visual Dinâmico**: Um visualizador de espectro de áudio em tempo real que reage às batidas da música com cores vibrantes.
- **📂 Carregamento Inteligente**: O app lembra da última pasta aberta, carregando sua biblioteca instantaneamente ao iniciar.
- **🎬 Suporte a Vídeo**: Além de um player de música, o AlexMusic também reproduz seus vídeos com controles imersivos.
- **🔁 Modos de Reprodução**: Suporte completo a Shuffle (Aleatório) e Repeat (Repetir Uma/Todas).
- **🌗 Tema Dinâmico**: Interface otimizada para modo escuro com acentos em Ciano e Pink Neon.

---

## 🚀 Tecnologias Utilizadas

- **Linguagem**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose) (Material 3)
- **Media Engine**: [AndroidX Media3](https://developer.android.com/guide/topics/media/media3) (ExoPlayer)
- **State Management**: Kotlin State & Flow
- **Permission Management**: Activity Result APIs para permissões de áudio e arquivos.

## 🛠️ Como Instalar e Rodar

1. Clone este repositório:
   ```bash
   git clone https://github.com/Viper0333/android-media-player.git
   ```
2. Abra o projeto no **Android Studio (Hedgehog ou superior)**.
3. Certifique-se de ter o **SDK 34** instalado.
4. Conecte seu dispositivo físico ou use um emulador.
5. Clique em **Run** ou execute via terminal:
   ```bash
   ./gradlew assembleDebug
   ```
